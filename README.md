# FedLD: Protecting Privacy and Enhancing Utility in Heterogeneous Graph Federated Learning

This repository provides a reference implementation of FedLD (Federated Logical Distillation) for heterogeneous Graph Federated Learning. 

Acknowledgment. The IDT implementation is adapted from Pluska et al., Logical Distillation of Graph Neural Networks (code: https://github.com/lexpk/LogicalDistillationOfGNNs
). Please retain their attribution when using or redistributing.
